# Raspberry Pi Pico Tetris

Simple Tetris using graphics backbuffer technique.

Dependencies

- adafruit/Adafruit BusIO
- adafruit/Adafruit GFX Library
- adafruit/Adafruit ST7735 and ST7789 Library

Hardware

- 1 Raspberry Pi Pico
- 1 TFT 128x128 SPI Panel
- 1 Stick module
- 2 push buttons
